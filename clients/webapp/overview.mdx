**The Hoop web app is where you manage connections, access policies, and sessions.** Available at [use.hoop.dev](https://use.hoop.dev) for managed accounts, or your self-hosted URL.

## Key Features

<CardGroup cols={2}>
  <Card title="Connection Management" icon="plug">
    Create connections to databases, servers, and cloud services with access settings and security policies in one place.
  </Card>

  <Card title="Access Control" icon="lock">
    Group-based permissions synced from your identity provider.
  </Card>

  <Card title="Guardrails" icon="shield-check">
    Pattern-based rules that block risky commands before they execute.
  </Card>

  <Card title="Security Features" icon="shield">
    Data masking, access requests, and session monitoring.
  </Card>
</CardGroup>

## Core Components

### Dashboard
Central hub for connections and activity:
- All connections at a glance
- Activity monitoring and alerts
- Recent session history

### Connection Manager
Supports databases (PostgreSQL, MySQL, MongoDB), cloud services (AWS, Kubernetes), web apps, and development environments.

### Access Control Panel
- User and group management
- Role-based access control
- Identity provider integration
- Policy configuration

### Session Monitor
- Live session visibility
- Session recordings and playback
- Audit logs and activity analytics
