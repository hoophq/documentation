One-Time Access Requests provide real-time oversight of commands and queries before execution. This guide will help you set up and configure command approvals for your connections.

## Enabling One-Time Access Requests

<Steps>
  <Step title="Access Connection Settings">
    Navigate to the connection you want to configure and select the "Additional Configuration" tab
  </Step>

  <Step title="Enable Access Requests">
    Toggle on the Access Requests switch to enable the approval workflow
  </Step>

  <Step title="Select Approval Groups">
    Choose which groups:
    - Need their commands approved
    - Can approve commands
    - Are exempt from approvals
  </Step>
</Steps>

### Access Levels
<AccordionGroup>
  <Accordion title="Approver Groups">
