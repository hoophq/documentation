title: "AWS Cognito"
---

## Requirements

- An AWS account
- `API_URL` is the public DNS name of the hoop gateway instance

Contact the hoop gateway administrator to retrieve the `API_URL` address.

<Steps titleSize="h3">
  <Step title="Create a User Pool">
    Go to AWS Management Panel > AWS Cognito > User Pools > Create User Pool

    - Select the checkboxes for **User Name** and **Email**, then click Next
    - In **Multi Factor Authentication**, select **No MFA** and click Next
    - In **Requirement Attributes**, select `name` and click Next
    - Configure **Message Delivery** and click Next
  </Step>
  <Step title="Integrate Your App Section">
    - Enter a name for the user pool (e.g., `hoopgateway`)
    - Select the checkbox **Use the Cognito Hosted UI**
    - Configure the Cognito domain

    <img src="/images/idp/aws-cognito-1.png" />

    - In **Initial App Client**, select **Public Client** and add an App client name
    - Select the option **Generate Client Secret**
    - Add the callback URL: `{API_URL}/api/callback`

    <img src="/images/idp/aws-cognito-2.png" />

    - In **Advanced app client settings**, select the checkbox **ALLOW_USER_PASSWORD_AUTH**
    - Configure the scopes: Email, OpenID, and Profile

    <img src="/images/idp/aws-cognito-3.png" />

    - Leave remaining options as default, click Next, and create the user pool
  </Step>
  <Step title="Collect the Credentials">
    - Go to the created user pool
    - Select the **App Integration** tab > click on `hoopauth`
    - Copy the **Client ID** and **Client Secret**

    <img src="/images/idp/aws-cognito-4.png" />
  </Step>
  <Step title="Collect Issuer Information">
  The URL is composed using the user pool ID: `https://cognito-idp.<aws-region>.amazonaws.com/<user-pool-id>`
  </Step>
</Steps>
