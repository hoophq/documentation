## Configuração

### String de Conexão
```
mongodb:&#x2F;&#x2F;username:password@host:port&#x2F;database
```

### Comando de Setup
```bash
hoop admin add resource-name --type mongodb --agent &lt;agent-name&gt; -e CONNECTION_STRING&#x3D;&#39;mongodb:&#x2F;&#x2F;user:pass@host:27017&#x2F;dbname&#39;
```
