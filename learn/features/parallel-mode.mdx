Hoop.dev's Web Terminal is a powerful web interface for managing secure infrastructure access. It allows you to connect to your resources, execute commands, and view the output in a web browser, also supports parallel execution of commands.

### Key Features

- **Simplified execution flow**: Run multiple resources simultaneously with a redesigned, faster, and more intuitive experience.
- **Smarter selection & control**: Access from Web Terminal or Runbooks to easily search, filter, or select by resource type with instant prefill.
- **Clear visibility & feedback**: Monitor with the new Execution Summary and identify statuses: Success, Error, or Approval Required  all in a single, organized view.

<Note>
  This feature is currently in Beta. Update your Hoop.dev version to the latest to get access to this feature.
</Note>

## Configuration

No additional configuration is required to use this feature. Go to the Web Terminal and access the new Parallel Mode on the top right corner to run multiple commands simultaneously.
