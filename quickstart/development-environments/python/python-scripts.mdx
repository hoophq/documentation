## Configuration

```bash
hoop admin create conn python3 -- python3
```

## Connect It

Now, itâ€™s possible to execute Python scripts straight from hoop.dev.

```bash
hoop exec python3 -i 'import os; print(os.environ)'
```

```bash
hoop exec python3 <<EOF
s3 = boto3.client('s3')
response = s3.list_buckets()

print('Existing buckets:')
for bucket in response['Buckets']:
    print(f'  {bucket["Name"]}')
EOF
```

Execute scripts directly

```bash
hoop exec python3 -- /tmp/myscript.py
```
