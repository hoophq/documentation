## Options

- Managed by hoop.dev. [Create a free account now](https://use.hoop.dev/register)
- Deploy and manage your own hoop.dev instance by following this guide.

## Get started

Installing hoop.dev in your infrastructure is straightforward. After everything is properly installed and configured, you will have a web interface to create and manage access to your connections for everyone you invite to use it.
