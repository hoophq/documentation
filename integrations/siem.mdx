---
title: "SIEM"
description: "Forward session events to your SIEM or logging pipeline using runbook hooks."
---

Hoop can forward session events — opens, closes, reviews — to any external system, making it straightforward to feed audit data into a SIEM, alerting platform, or custom logging pipeline.

The recommended approach uses **runbook hooks**: lightweight scripts in your runbooks git repository that execute automatically when session events occur. No third-party webhook service is required.
