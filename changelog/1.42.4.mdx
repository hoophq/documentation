### Improvement

**Simplified Redis Connection Setup ([#60](https://github.com/hoophq/documentation/pull/60))**

Redis connections no longer require username and password fields. The `--user` and `-a` parameters are removed from the default configuration.

**What changed:**
- Removed mandatory `USER` and `PASSWORD` fields from the Redis connection form
- `redis-cli` command now requires only host, port, and database number
- Faster setup for dev and local Redis instances without authentication

---

Less friction, faster connections.
