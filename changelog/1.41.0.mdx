---

### TL;DR
- **New:** Redesigned Postgres native connection experience; connectivity check for resources.
- **Security:** Auth tokens moved from URL callbacks to cookies.
- **Fixed:** Agent Daemon now finds local commands correctly; hoop key persists across VM restarts.

---

### New Features

- **Postgres native connections**: Redesigned connection experience.
  [PR #1036](https://github.com/hoophq/hoop/pull/1036) and [PR #998](https://github.com/hoophq/hoop/pull/998)

- **Resource connectivity check**: Verify connectivity before using a resource.
  [PR #1020](https://github.com/hoophq/hoop/pull/1020) and [PR #1027](https://github.com/hoophq/hoop/pull/1027)

---

### Security

**Auth tokens removed from URL callbacks.** Tokens now use cookies, eliminating exposure in server logs and browser history.
[PR #1034](https://github.com/hoophq/hoop/pull/1034)

---

### Bug Fixes

- Fixed the Agent Daemon failing to find local commands on the installed machine.
  [PR #1029](https://github.com/hoophq/hoop/pull/1029)

- The hoop key now persists across VM restarts for easier management.
  [PR #1029](https://github.com/hoophq/hoop/pull/1029)

---

### Miscellaneous
- Removed the `API_URL` environment variable requirement across all environments.
  [PR #1028](https://github.com/hoophq/hoop/pull/1028)
