### TL;DR
- **New:** Opt-in anonymous tracking; Command Palette (`Cmd/Ctrl+K`); Agent Daemon Mode
- **Fixed:** MySQL session kill; MongoDB schema; API key propagation; Jira re-run

---

### New Features

#### Opt-in Anonymous Tracking
Admins see a consent prompt to enable anonymous tracking. No identifiable data is collected, only product analytics.

[PR #1007](https://github.com/hoophq/hoop/pull/1007) and [PR #1011](https://github.com/hoophq/hoop/pull/1011)

#### Command Palette
Hit `Cmd+K` on macOS or `Ctrl+K` on Windows/Linux to search and navigate the Hoop web app.

[PR #1008](https://github.com/hoophq/hoop/pull/1008)

#### Agent Daemon Mode
Run the Hoop Agent as a background daemon with a dedicated management command.

[PR #997](https://github.com/hoophq/hoop/pull/997)

---

### Bug Fixes

- Fixed MySQL sessions failing to kill (community report)
- Fixed terminal connections list not scrolling
- Fixed API key propagation dropping calls
- Fixed MongoDB schema tree browsing errors
- Removed “Re-run session” option from Jira integrations
- Improved MongoDB integration stability across edge cases
